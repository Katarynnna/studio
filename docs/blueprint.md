# **App Name**: Trail Angel Hub

## Core Features:

- Interactive Trail Map: Display static locations of Trail Angels, filterable by name, location, hiker's current location, dates, donation expected and services offered: rides, Bathroom, Private Room, Bed(s), Couch/Floor, Camping, Kitchen, Meals, Secure storage, address for deliveries (bailment), Laundry, WiFi / Internet. Show PCT, AT, CDT trail tracks on the map
- Trail Angel Card: Show Logistics Dashboard upon pin click, including service icons, availability calendar, location, about me, gallery, badges, reviews and gated contact info, similar to a Warmshowers profile
- Personalized Messaging: Enable hikers to send personalized messages to Trail Angels to coordinate logistics, with optional verification for higher success rates.
- Availability Calendar: Dynamic calendar for Trail Angels to manage availability and set active/inactive dates.
- Reciprocal Review System: Allow both hikers and Trail Angels to leave feedback, building a reputation economy.
- Geo-fenced Trail Radio: A side feed displaying dated comments relevant to the current map view; hikers can post utility requests.
- Content Moderation: AI tool that identifies posts containing inappropriate language or suspicious requests.

## Style Guidelines:

- Primary color: Forest green (#4CAF50), reminiscent of lush trails, symbolizes growth and nature.
- Background color: Soft off-white (#F8F8F8) to provide a clean, unobtrusive backdrop that emphasizes the content, akin to paper maps.
- Accent color: Sunset orange (#FF9800), complementary to green, conveys energy and enthusiasm.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern look, is easy to read.
- Code font: 'Source Code Pro' for any displayed code (e.g., error messages).
- Use line-style icons to represent services like rides, showers, food, etc.
- Prioritize map and actionable content.
